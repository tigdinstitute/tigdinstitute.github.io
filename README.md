# tigdinstitute GitHub Pages Site

This is a simple static website for the tigdinstitute organization.

## Publish to GitHub Pages

1. Create a repository under the organization `tigdinstitute` with the name:
   - `tigdinstitute.github.io`

2. Clone the repository locally:
   ```bash
   git clone https://github.com/tigdinstitute/tigdinstitute.github.io.git
   cd tigdinstitute.github.io
   ```

3. Copy these files into the repository root, then commit and push:
   ```bash
   git add .
   git commit -m "Initial GitHub Pages site"
   git push origin main
   ```

4. Verify GitHub Pages is set to publish from the `main` branch root.

## Site URL

After publishing, the site will be available at:

- `https://tigdinstitute.github.io`

## Notes

- Use this site as a starting point. You can update text, links, and branding.
- If you want a custom domain later, add it in the repository settings under Pages.
